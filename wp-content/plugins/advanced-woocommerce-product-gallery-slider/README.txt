== Description ==

Instantly transform the gallery on your WooCommerce Product page into a fully Responsive Stunning Carousel Slider.

<br />
=  Plugin Features =

<li>Gallery Layout [Horizontal/Vertical Silder]</li>
<li>Responsive Layout</li>
<li>Very Lightweight</li>
<li>Easy Admin Settings</li>
<li>Navigation support</li>
<li>Slider AutoPlay Options</li>
<li>Customize Slider Arrow (Color Options)</li>
<li>Attractive Lightbox effect</li>
<li>Working with Most of Premium themes</li>
<li>Touch and Swipe enabled etc.</li>

Feel free to contact me on unikinfotech.web@gmail.com.


== Installation ==

1. Upload `woo-product-gallery-image-slider` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
